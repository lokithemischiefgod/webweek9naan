Cara Kerja Internet Secara Umum

Internet adalah jaringan global yang menghubungkan komputer di seluruh dunia. Jaringan ini bekerja dengan cara mengirim dan menerima data melalui protokol komunikasi seperti TCP/IP. Protokol TCP/IP memungkinkan komputer untuk berkomunikasi satu sama lain meskipun menggunakan perangkat keras yang berbeda.

Referensi: Computer Hope - Internet

Definisi dan Cara Kerja HTTP & HTTPS

HTTP (Hypertext Transfer Protocol) adalah protokol yang digunakan untuk mentransfer data di web. HTTPS adalah versi aman dari HTTP yang menggunakan enkripsi SSL/TLS untuk melindungi data yang dikirim antara klien dan server. HTTPS memastikan integritas dan privasi data yang dikirim, terutama untuk transaksi sensitif seperti pembayaran online.

Referensi: MDN Web Docs - HTTP Overview

Definisi dan Cara Kerja Nama Domain

Nama domain adalah alamat unik yang digunakan untuk mengidentifikasi situs web di internet. Nama domain dipetakan ke alamat IP oleh DNS agar mudah diakses oleh pengguna. Contoh nama domain: www.example.com.

Referensi: Webopedia - Domain Name

Definisi dan Cara Kerja Hosting

Hosting adalah layanan yang menyediakan ruang server untuk menyimpan data situs web agar dapat diakses melalui internet. Penyedia hosting mengelola server dan memastikan situs web selalu tersedia online.

Referensi: HostGator - What is Web Hosting?

Definisi dan Cara Kerja DNS

DNS (Domain Name System) adalah sistem yang digunakan untuk menerjemahkan nama domain ke alamat IP yang dapat dikenali oleh komputer. DNS berfungsi seperti buku telepon untuk internet.

Referensi: Cloudflare - What is DNS?

Definisi dan Cara Kerja Browser

Browser adalah aplikasi yang digunakan untuk mengakses dan menampilkan halaman web di internet. Browser berkomunikasi dengan server menggunakan HTTP/HTTPS untuk mengunduh halaman web dan menampilkannya kepada pengguna. Contoh browser: Google Chrome, Mozilla Firefox, Microsoft Edge.

Referensi: How-To Geek - What is a Web Browser?
